## インストール
今のところEclipseでの開発。[モジュールパスを追加する](https://github.com/LWJGL/lwjgl3-wiki/wiki/1.2.-Install#eclipse)必要がある。

Mavenは遅いという感想しかもっていないため使わない。


必要なjarはlwjgl.userlibrariesを参照。
[ここ](https://www.lwjgl.org/customize)からダウンロード可。

（jarとダウンロードページの名前が一致してない可能性あり、うまく書く方法不明）
